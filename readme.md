##14.chmod

	برای مجوز دادن

	###انواع مجوز ها
		اگر از دستور
		'ls -l'
		برای نمایش جزییات استفاده کنیم با چیزی شبیه موارد زیر روبه رو خواهیم شد


	  	"-rw-r--r-- 1 ali ali 120 Jun 10 notes.txt"
		قسمت موردنظرمون این تیکه هستش
		"-rw-r--r--"
		اگه ما این تیکه رو به سه قسمت مساوی تسیم کنیم, هرقسمت نشان دهنده مجوز های هر گروه هستش که در ادامه به آن خواهیم پرداخت
		
		1. r = read    خواندن
		2. w = write   نوشتن/تغییر دادن
		3. x = execute اجرا کردن
		
		و سه گروه عبارتند از
		
		1. owner
		2. group
		3. others
		
		برای مثال عبارت بالا یعنی
		
		owner:  rw-  می‌تواند بخواند و بنویسد
		group:  r--  فقط می‌تواند بخواند
		others: r--  فقط می‌توانند بخوانند
		
	###طریقه دادن مجوز
		
	    **1.**

		راه اول از طریق حروف هست
		فرم ا	صلی کامتد به این صورت است
‍‍		```	‍‍‍
			chmod (u/g/o/a) +/- rwx(مجوز ها) name.sh
 		```

		u = (User/Owner): مالک فایل
		g = (Group): گروه
		o = (Others): سایرین
		a = (All): همه (هر سه مورد بالا
		
	    **2.**

		روش دوم روش اعداد هست که اینطوریه که یک عدد سه رقمی داریم که جایگاه هر عدد نشان دهنده نوع کاربر هست و مجموع هر عدد 
		نشان دهنده مجوز ها است
		مثال

	‍‍```
			chmod 755 hello.sh (دسترسی کامل مالک، بقیه فقط بخوانند و اجرا کنند (استاندارد اسکریپت‌ها))
			chmod 644 hello.sh (مالک بخواند و بنویسد، بقیه فقط بخوانند (استاندارد فایل‌های متنی))
			chmod 700 hello.sh (فقط مالک دسترسی کامل داشته باشد و بقیه هیچ (خصوصی))
	```
		
		4 = خواندن (Read)
		2 = نوشتن (Write)
		1 = اجرا کردن (Execute)
		0 = هیچ دسترسی
