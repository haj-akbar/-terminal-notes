# Terminal Learning Journey 🐧

این خلاصه من برای دستورات **ترمینال لینوکس **است

## 0.The typical command format

شکل رایج کامند ها  
	

command [option] [target]

## 1. pwd
مسیر فعلی را نشان می‌دهد (**Print Working Directory**).

### تفاوت مسیر مطلق و نسبی
- **مسیر مطلق (Absolute Path):** از ریشه سیستم شروع می‌شود.
- **مسیر نسبی (Relative Path):** نسبت به مکانی که الان در آن هستید سنجیده می‌شود.

---

## 2. cd
برای جابه‌جایی بین پوشه‌ها به کار می‌رود (**Change Directory**).

### موارد پرکاربرد
1. `.` → اشاره به پوشه فعلی
2. `..` → رفتن به یک پوشه عقب‌تر
3. `~` → رفتن به پوشه Home کاربر
4. `/` → رفتن به ریشه سیستم

---

## 3. ls
برای مشاهده فایل‌ها و پوشه‌ها استفاده می‌شود (**List**).

### گزینه‌های پرکاربرد
- `ls -l` → نمایش جزئیات بیشتر
- `ls -a` → نمایش فایل‌های مخفی
- `ls -la` → نمایش همه چیز همراه با جزئیات

معمولاً `ls -la` خیلی پرکاربرد است.

---

## 4. mkdir
برای ساخت پوشه جدید استفاده می‌شود (**Make Directory**).


`mkdir folder1 folder2 folder3`

### گزینه‌ها
- `-p` → برای ساخت پوشه‌های تو در تو


`mkdir -p me/university/homework`

---

## 5. touch
برای ساخت فایل خالی استفاده می‌شود.


`touch name1.md name2.txt name3.sh`

اگر بخواهیم فایل را در یک مسیر مشخص بسازیم:


`touch me/university/homework/project1.txt`

---

## 6. echo
برای چاپ متن یا نوشتن داخل فایل استفاده می‌شود.


`echo "متن مورد نظر" > name.txt`

اگر داخل فایل چیزی باشد، با این دستور پاک می‌شود و متن جدید جایگزین آن می‌شود.

اگر بخواهیم متن به انتهای فایل اضافه شود، باید از `>>` استفاده کنیم:


`echo "متن جدید" >> name.txt`

---

## 7. cat
برای نمایش سریع محتوای فایل‌ها استفاده می‌شود.  
برای فایل‌های کوچک مناسب‌تر است.

---

## 8. less
برای خواندن فایل‌های بزرگ مناسب است.

### کلیدهای پرکاربرد در less
- `Space` → صفحه بعد
- `b` → صفحه قبل
- `/word` → جست‌وجوی کلمه
- `n` → رفتن به نتیجه بعدی
- `q` → خروج

---

## 9. cp
برای کپی کردن فایل یا پوشه استفاده می‌شود (**Copy**).


`cp name.txt docs/new-name.txt`

اسم جدید اختیاری است.

### گزینه‌ها
- `-r` → برای کپی کردن پوشه و محتویات آن

---

## 10. mv
برای جابه‌جا کردن یا تغییر نام فایل و پوشه استفاده می‌شود (**Move**).


`mv name1.txt name2.txt`
`mv task .`

- دستور اول برای تغییر نام است.
- دستور دوم پوشه یا فایل `task` را به مسیر فعلی منتقل می‌کند.

---

## 11. rm
برای حذف فایل یا پوشه استفاده می‌شود (**Remove**).


`rm text.txt docs/q.md`

### گزینه‌ها
- `-r` → حذف پوشه و تمام محتویات آن
- `-i` → گرفتن تأیید قبل از حذف
- `-rf` → حذف اجباری و بدون پرسش
- `rmdir` → حذف پوشه خالی

> `rm -rf` دستور خطرناکی است و باید با احتیاط استفاده شود.

---

## 12. grep
برای پیدا کردن یک کلمه یا عبارت داخل فایل‌ها استفاده می‌شود.


`grep "word" name.txt`

### گزینه‌ها
- `-i` → بدون حساسیت به بزرگی و کوچکی حروف
- `-n` → نمایش شماره خط
- `*.txt` → جست‌وجو در چند فایل متنی
- `-r` → جست‌وجوی بازگشتی در پوشه‌ها

---

## 13. find
برای پیدا کردن فایل‌ها و پوشه‌ها استفاده می‌شود.


`find . -name "test.txt"`

### گزینه‌ها و مثال‌ها
- `-type` → مشخص کردن نوع
- `-name` → جست‌وجو بر اساس نام


`find . -type f -name "*.txt"`

یعنی از پوشه فعلی شروع کن، فقط فایل‌ها را پیدا کن، و فقط آن‌هایی را نشان بده که با `.txt` تمام می‌شوند.


`find me/university -name "todo.txt"`

یعنی در مسیر `me/university` فایل `todo.txt` را پیدا کن.

---

## 14. chmod
برای تغییر مجوزهای فایل‌ها و پوشه‌ها استفاده می‌شود (**Change Mode**).

### انواع مجوزها
اگر از دستور زیر استفاده کنیم:


ls -l

ممکن است خروجی چیزی شبیه این باشد:


-rw-r--r-- 1 ali ali 120 Jun 10 notes.txt

بخش مهم برای ما این قسمت است:


-rw-r--r--

اگر این بخش را به سه قسمت تقسیم کنیم، هر قسمت نشان‌دهنده مجوزهای یک گروه است.

### نوع مجوزها
1. `r` = خواندن (**Read**)
2. `w` = نوشتن (**Write**)
3. `x` = اجرا کردن (**Execute**)

### گروه‌ها
1. **owner** → مالک
2. **group** → گروه
3. **others** → سایرین

مثال بالا یعنی:

- **owner:** `rw-` → می‌تواند بخواند و بنویسد
- **group:** `r--` → فقط می‌تواند بخواند
- **others:** `r--` → فقط می‌توانند بخوانند

### روش‌های دادن مجوز

#### 1. روش حروفی

فرم کلی دستور:


`chmod [u/g/o/a][+/-][rwx] name.sh`

معنی حروف:
- `u` = مالک (**User / Owner**)
- `g` = گروه (**Group**)
- `o` = سایرین (**Others**)
- `a` = همه (**All**)

مثال:


`chmod u+x hello.sh`

یعنی به مالک اجازه اجرا بده.

---

#### 2. روش عددی

در این روش از یک عدد سه‌رقمی استفاده می‌شود.  
هر رقم نشان‌دهنده مجوزهای یکی از گروه‌ها است.

مقدارها:
- `4` = خواندن
- `2` = نوشتن
- `1` = اجرا
- `0` = بدون دسترسی

مثال‌ها:


`chmod 755 hello.sh`
`chmod 644 hello.sh`
`chmod 700 hello.sh`

توضیح:
- `755` → مالک دسترسی کامل دارد، بقیه فقط خواندن و اجرا
- `644` → مالک خواندن و نوشتن دارد، بقیه فقط خواندن
- `700` → فقط مالک دسترسی کامل دارد

---

## 15. man

`man command`

مثال:


`man ls`

یعنی راهنمای دستور `ls` را نشان بده.
